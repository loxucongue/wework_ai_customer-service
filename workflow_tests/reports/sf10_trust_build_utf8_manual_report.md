# SF10 信任建立 UTF-8 测试人工结论

测试时间：2026-05-22

## 结论

SF10 信任建立已完成 24 条 UTF-8 测试。接口错误 0 条，空回复 0 条。

整体表现明显好于 SF6/SF8/SF9：多数信任问题能识别为 `trust_issue`，能正确输出 `image` 类型图片，没有暴露“知识库/检索/工具”等内部词，也没有明显“百分百安全/保证有效”的强违规表达。

但当前仍不建议直接上线，主要问题是转人工边界失败、图片发送过多、部分回复承诺感偏强，以及个别资料和客户问题不够匹配。

## 数据概览

- 总用例：24
- 接口错误：0
- 空回复：0
- `trust_issue`：21
- `human_request`：1
- `complaint_refund`：1
- `image_inquiry`：1
- 触发 image 消息：19 条
- 内部词暴露：0
- 明显绝对效果/安全承诺：0

## 表现较好的点

- 正规性、产品真假、设备、医生/技师、售后、消毒流程等问题基本能触发图片资料。
- 图片消息使用 `type=image`，不是把 URL 塞进文本。
- 多数回复有先接住顾虑，再给流程或资料的结构。
- AI/真人问题处理比较克制，没有假装真人。
- 没有出现“知识库显示、工具返回、检索结果”等客户侧禁词。

## 主要问题

1. 明确要真人没有转人工  
   `SF10_015` 客户说“你能不能让真人顾问跟我说”，识别为 `human_request`，但回复成普通欢迎语。  
   正确应直接转人工或回复“我帮你安排顾问接上”。

2. 投诉风险没有转人工  
   `SF10_023` 客户说“你们别骗我，我要投诉”，识别为 `complaint_refund`，但回复成普通欢迎语。  
   这是强制转人工场景，不能继续普通信任建立。

3. 图片发送过多  
   `SF10_012` 售后问题一次发了 2 张图片。  
   `SF10_024` “先发点资料”也一次发 2 张图片。  
   之前设计规则是每轮最多 1 张最相关图片，避免刷屏和信息不聚焦。

4. 承诺感偏强  
   多条回复出现类似：
   - “确保用到的产品都是正品”
   - “不会有隐形消费”
   - “不会有额外费用”
   - “你可以放心”
   - “保障安全”
   
   更稳妥表达应是“到店可核对产品来源”“费用会提前确认”“帮助降低风险”，不要做绝对化承诺。

5. 资料和问题不完全匹配  
   `SF10_009` 客户怕“不自然、翻车”，系统发术后护理图，但更应该解释方案审美、面诊评估、预期沟通，不应把护理图当成解决“翻车”的主要证据。  
   `SF10_003` 客户问资质，回复后又说“不会有额外收费”，话题跳到了价格透明，不够连贯。

6. 可能编造医生/设备细节  
   `SF10_006` 提到“这两位医生和技师都有多年从业经验，眼部和皮肤项目都有专业擅长方向”。  
   如果这些不是图片知识库明确返回的字段，就属于风险表达。医生/技师年限、擅长方向必须来自资料。

7. 案例咨询边界不足  
   `SF10_018` 客户问真实案例，被路由成 `image_inquiry`，回复“有一些经过隐私处理的真实案例可以参考”。  
   案例必须有授权和素材库结果，不应口头承诺有真实案例。

## 必须调整的规则

1. 强制转人工优先级  
   以下命中后，SF10 不得继续普通回复：
   - 明确要求真人/顾问
   - 投诉、退款、维权、曝光
   - 强烈不满或纠纷

2. 每轮最多 1 张图片  
   图片必须选择与当前顾虑最相关的一张：
   - 正规性：资质/许可
   - 产品真假：溯源/防伪
   - 设备：设备授权/检测
   - 医生技师：资质资料
   - 售后：售后流程
   - 消毒安全：消毒/操作流程

3. 禁止承诺式表达  
   禁止：
   - 确保正品
   - 绝对安全
   - 可以放心
   - 不会有隐形消费
   - 不会有额外费用
   - 保证效果
   
   建议改为：
   - 到店可核对产品来源
   - 项目和费用会提前确认
   - 操作前会评估并说明风险
   - 不适合的情况不建议做

4. 不得扩展图片资料外的信息  
   图片或资料没有明确写医生年限、设备来源、门店规模、售后承诺时，回复不能补充这些细节。

5. 案例必须依赖授权素材  
   没有授权案例库结果时，只能说“案例资料需要确认授权后提供”，不能说“我们有真实案例”。

## 建议安全模板

正规性质疑：
“你有这个顾虑很正常，医美机构确实要先看资质。我先发一张资质资料给你参考，到店也可以进一步核对。”

产品真假：
“产品来源这块可以到店核对包装和溯源信息，我先发你溯源流程参考。具体使用产品会在方案确认时说清楚。”

怕推销：
“可以只先了解和面诊，不需要当天决定。方案和费用会先说明，你接受后才继续安排。”

要求真人：
“可以，我帮你安排顾问接上。”

投诉：
“我理解你的担心，这个我帮你转人工同事处理。”
